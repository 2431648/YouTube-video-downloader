# YouTube Video Downloader (BCA Mini Project)

This is a web-based YouTube Video Downloader built using Python (Flask) and Pytube.

## Features
- Clean Dark UI with Hover Effects
- Downloads videos in Highest Resolution
- Custom "Glassmorphism" Design

## How to Run on Your Computer
1. Install Python.
2. Download this code as a ZIP.
3. Open terminal and run:
   ```bash
   pip install flask pytubefix
   python app.py
